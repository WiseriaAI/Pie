# v0.11.1 — Centered greeting empty state

打开 side panel 时不再展示 READY 标签 + skill 推荐卡，改为垂直居中的随机问候语 + 副标题。7 个中英双语变体，每次 mount 随机选一条；切换语言会翻译同一条，不重新抽。

## What changed

- **新空状态**：READY 标签 + 标题 + skill suggestion 卡 → 居中问候语 + 单行副标题
- **7 个 greeting 变体**：中英双语朋友型温度文案，mount 时随机
- **Locale 稳定**：useMemo 锁定 greeting key，切语言只 re-translate，不重新抽

## Why

- skill 推荐卡在用户已经知道用什么 skill 后只是噪音
- 垂直居中 + 减卡片让首屏更克制，与 BYOK 工具调性更一致

## 不变

- 已有 session / skill / instance 不受影响
- 无 manifest 权限变化，用户透明升级
